# OPGG

### - Deploy URL
http://opgg-bucket.s3-website.ap-northeast-2.amazonaws.com/

<br>

### - Skills
```
React, Redux, Redux-thunk, React-intersection-observer, Styled-components ...
```

<br>

## # 실행 방법

### - Clone repository
```
git clone https://github.com/seungdeng17/opgg.git
```

### - Install dependencies
```
npm install
```

### - Run project
```
npm start
```
